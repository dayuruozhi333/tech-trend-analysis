# 基于LDA模型的文献主题分析系统

## 1. 核心概述
- **目标**：利用 LDA 主题建模与大语言模型，高效解析海量学术论文的主题结构。
- **价值**：自动聚类文献主题、识别热点方向、生成 AI 辅助解读与报告。
- **用户**：高校科研人员、课题组、信息分析师、技术决策者。

## 2. 架构与技术
- **前端**：React + TypeScript + Vite，Plotly.js 支撑主题概览、趋势曲线、关键词图谱等可视化。
- **后端**：Flask 提供 REST API，封装主题检索、趋势分析、AI 报告等服务。
- **模型与数据**：Gensim LDA、NLTK、spaCy，处理 CSV / Parquet / JSON 数据并维护 15 个主题簇。
- **部署环境**：Python 3.10+、Node.js 18+，兼容信创 ARM64 平台。

## 3. 核心功能
- **主题发现**：多轮文本清洗、关键词抽取、主题标签生成与相似主题对齐。
- **主题洞察**：关键词权重、主题贡献度、文献分布、主题关系图谱。
- **趋势分析**：年度热度聚合、主题演化、交互式趋势对比。
- **AI 洞察**：调用 ModelScope API，生成主题综述、趋势解读、地图视角与综合分析报告。

## 4. 工程特点
- **性能实践**：分块读写、懒加载、缓存策略，支撑 50 万篇文献处理与实时查询。
- **可视化体验**：主题卡片、趋势曲线、关键词气泡图、主题关系网络，桌面与移动端自适应。
- **信创适配**：验证飞腾 D3000 + 银河麒麟环境，提供依赖安装与性能指标。

## 5. 启动步骤
```bash
python start/start.py        # 启动前后端
# 或分别启动
cd backend && python run.py
cd frontend && npm run dev
```
访问 `http://localhost:5173` 浏览主题、查看趋势并触发 AI 分析。

## 6. 项目结构
```
frontend/        # React 前端
backend/         # Flask 后端
data_processing/ # 数据预处理与模型训练
start/           # 启动脚本
```

## 7. 项目亮点
- LDA 与大模型结合，实现主题发现、趋势评估、洞察生成的一体化分析。
- 一键启动、跨平台兼容、轻量部署，适配信创环境。
- 面向科研与情报场景的中文化可视化与交互体验。
