## 响应式设计适配

### 桌面端 (≥1200px)
```
┌─────────────────────────────────────────────────────────────────┐
│ 侧边栏(220px) │ 主内容区域(自适应)                              │
└─────────────────────────────────────────────────────────────────┘
```

### 平板端 (768px-1199px)
```
┌─────────────────────────────────────────────────────────────────┐
│ 可折叠侧边栏 │ 主内容区域(自适应)                               │
└─────────────────────────────────────────────────────────────────┘
```

### 移动端 (<768px)
```
┌─────────────────────────────────────────────────────────────────┐
│ 汉堡菜单按钮 │ 主内容区域(全宽)                                │
└─────────────────────────────────────────────────────────────────┘
```
