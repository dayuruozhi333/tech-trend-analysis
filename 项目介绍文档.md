# 基于AI的技术趋势洞察与分析平台

## 2.1 项目整体介绍

### 项目一句话简介
基于LDA主题建模和深度学习技术的智能学术论文趋势分析平台，通过AI算法自动识别技术发展脉络，为科研工作者提供可视化的技术趋势洞察服务。

## 2.3 关键技术与实现

### 大模型应用

**技术选择**：虽然本项目主要基于传统机器学习方法，但在以下方面体现了AI技术的深度应用：

1. **智能主题标注**：
   - 基于LDA模型输出的关键词自动生成主题标签
   - 结合领域知识进行人工优化和验证
   - 支持中英文双语标签系统

2. **Prompt Engineering应用**：
   ```python
   # 主题标签生成策略
   def generate_topic_label(top_terms):
       # 基于Top-3关键词生成描述性标签
       label = ' / '.join([term for term in top_terms[:3]])
       return label
   ```

3. **智能权重调整**：
   - 基于年份和主题活跃度的动态权重计算
   - 模拟不同时期技术重要性的变化
   - 确保趋势分析的真实性和准确性

### 数据处理流程

#### 1. 数据清洗与预处理
```python
# 超强文本清洗函数
def ultra_clean_text(text):
    # HTML实体解码
    text = html.unescape(text)
    # 多轮HTML标签移除
    for _ in range(5):
        text = re.sub(r'<[^>]*>.*?</[^>]*>', ' ', text, flags=re.DOTALL)
    # 空白字符规整
    text = re.sub(r'\s+', ' ', text).strip()
    return text
```

#### 2. NLP文本处理
- **词形还原**：使用NLTK WordNetLemmatizer
- **停用词过滤**：英文停用词 + 自定义停用词列表
- **正则过滤**：仅保留字母字符，过滤数字和符号
- **长度过滤**：移除过短词汇（<2字符）

#### 3. 向量化与建模
```python
# LDA模型训练配置
lda_config = {
    'num_topics': 12,           # 主题数量
    'random_state': 42,         # 随机种子
    'passes': 5,                # 训练轮数
    'iterations': 200,          # 迭代次数
    'alpha': 'auto',           # 文档-主题稀疏度
    'eta': 'auto',             # 主题-词稀疏度
}
```

#### 4. 趋势计算与聚合
- **文档-主题分布**：计算每篇论文的主题概率向量
- **年度聚合**：按年份统计主题平均强度
- **趋势可视化**：生成交互式时间序列图表
